# 🎂 Mona Homemadecakes — Luxury Cake Boutique Website

> A premium single-page website for a luxury homemade cake boutique in Namakkal, Tamil Nadu — crafted with pure HTML, CSS & Vanilla JavaScript.

🔗 **Live Demo:** 
https://monaahomemadecakes.online/


## ✨ Features

- 🎬 Animated page loader with branding
- 🖱️ Custom animated cursor with glow & ring effect
- 🌸 Floating emoji particles in the hero section
- 📜 Scroll progress bar at the top
- 🔝 Sticky navbar that changes style on scroll
- 🎂 Signature cake collections with 6 categories & pricing
- 🖼️ Gallery slider with fade effect & autoplay (Swiper.js)
- 💬 Testimonials carousel slider
- 🎉 Occasions section (Weddings, Birthdays, Corporate & more)
- 🕐 Live open/closed store status based on real clock time
- 👁️ Scroll-reveal animations on all sections
- 🔢 Animated stat counters
- 📍 Embedded Google Maps location
- 📱 Fully responsive with mobile hamburger menu
- 🃏 3D card tilt effect (desktop only)
- ✨ Click sparkle animation
- 💬 Floating WhatsApp & Call FAB buttons


## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Structure & markup |
| CSS3 | Styling, animations, glassmorphism |
| Vanilla JavaScript | All interactivity & effects |
| [Swiper.js v11](https://swiperjs.com/) | Gallery & testimonials sliders |
| Google Fonts | Cormorant Garamond, Cinzel, Lato |
| Google Maps Embed | Location section |
| Vercel | Deployment & hosting |

## 🚀 Getting Started

No build tools or installs needed — it's pure HTML/CSS/JS.

```bash
# 1. Clone the repository
git clone https://github.com/your-username/mona-homemadecakes.git

# 2. Open in browser
cd mona-homemadecakes
open index.html
```

Or just drag `index.html` into any browser.

## ☁️ Deployment

This project is deployed on **Vercel**.  
Every push to the `main` branch triggers an automatic re-deployment.

To deploy your own copy:
1. Fork this repo
2. Go to [vercel.com](https://vercel.com) → Import Project
3. Select the repo → Deploy ✅
